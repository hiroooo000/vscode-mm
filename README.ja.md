# VS Code Mind Map

[English](./README.md)

VS Code 内でアイデアを直感的に視覚化・整理。

この拡張機能は、VS Code に直接統合された強力で直感的なマインドマップエディタを提供します。グラフィカルな形式でアイデアの作成、編集、管理を行うことができます。

## ✨ 特徴

- **直感的な編集**: シンプルなキーボードショートカットや UI を使用して、ノードの追加、削除、変更が可能です。
- **画像の貼り付け (Image Paste)**: クリップボードから画像を直接ノードに貼り付けることができます。
- **MindElixir による強力な描画**: [MindElixir](https://github.com/ssshooter/mind-elixir) をコアエンジンとして採用し、スムーズなレンダリングと操作感を実現しています。
- **双方向同期**: ビジュアルマップと基となるドキュメントモデル（JSON）間のリアルタイム同期。
- **ネイティブな統合**: VS Code の Undo/Redo、ファイル保存、ワークベンチテーマとのシームレスな統合。

## 🎨 MindElixir について

本拡張機能は、マインドマップの描画および操作のコアライブラリとして **[MindElixir](https://github.com/ssshooter/mind-elixir)** を使用しています。

MindElixir は、柔軟で高速なマインドマップエンジンであり、本プロジェクトでは以下の役割を担っています：
- マインドマップのグラフィカルなレンダリング。
- ノードの展開・折りたたみ、ドラッグ＆ドロップなどのインタラクション。
- 標準的なマインドマップデータ形式の提供。

## 🚀 はじめに

1. **ファイルの作成**: `.mm` または `.mindmap` 拡張子を持つ新しいファイルを作成します。
2. **マインドマップで開く**: ファイルを右クリックして「エディターで開く (Open With...)」を選択し、「Mind Map」を選択します。
3. **マッピングの開始**: UI またはショートカットキーを使用してマインドマップを構築します。

### 画像の貼り付け方法
1. クリップボードに画像をコピーします。
2. マインドマップ上の対象ノードを選択します。
3. `Ctrl + V` (または `Cmd + V`) を押して画像をノードに貼り付けます。

## ⌨️ ショートカット

一般的なショートカット（MindElixir のデフォルトに準拠）:
- `Tab`: 子ノードを追加
- `Enter`: 兄弟ノードを追加
- `Delete`: ノードを削除
- `Space`: ノードテキストを編集

## 🛠️ 技術概要

- **コア**: VS Code Extension API の `CustomTextEditorProvider` を使用して構築。
- **Frontend**: Webview 内で **MindElixir** (JavaScript/TypeScript) を使用。
- **データ形式**: MindElixir と互換性のある JSON ベースの構造を採用。

## 🧪 開発

プロジェクトに貢献する場合、またはソースからビルドする場合：

### Dev Container

このプロジェクトは Dev Container で構成されています。
1. プロジェクトを Dev Container で再度開きます (Reopen)。
2. 環境と依存関係が自動的にセットアップされます。

```bash
# 依存関係のインストール
npm install

# プロジェクトのコンパイル
npm run compile

# テストの実行
npm run test:all:devcontainer
```
